Executive Summary

TeachMeFy is an AI-powered educational and utility bot for Discord that provides instant responses, explanations, and practical utilities in real time. It does not store user messages and prioritizes transparency, privacy, and responsible use.

What Is TeachMeFy?

TeachMeFy is a Discord bot that uses third-party AI services to deliver helpful replies, clarify concepts, and provide server utilities within Discord.

  • General learning and explanations
  • Curiosity-driven questions
  • Practical Discord-based utilities

Note: TeachMeFy is not a professional advisor and should not replace independent judgment.

Why TeachMeFy Exists

TeachMeFy lowers friction between curiosity and understanding by embedding AI-assisted help directly where conversations already happen: Discord.

AI-Powered Responses

Real-time answers generated by external AI APIs.

Learning-Focused Design

Simplifies complex topics to make concepts accessible.

Privacy-First

No message or user ID storage; inputs are processed and discarded.

How It Works

  1. A user sends a message or command to TeachMeFy.
  2. The input is processed in real time via external AI services.
  3. A response is generated and returned to the channel.
  4. No message or response is retained by TeachMeFy.

Transparency & Limitations

TeachMeFy is explicitly AI-powered. Outputs may be incorrect, incomplete, or biased and should not be treated as professional advice.

Responsible Use: TeachMeFy may not be used to harass, generate illegal content, or circumvent safeguards. Users must comply with Discord's rules and applicable laws.
